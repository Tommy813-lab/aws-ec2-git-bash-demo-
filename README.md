# AWS EC2 Deployment with Git Bash

## 🧠 Project Summary
This project demonstrates how to launch and connect to an EC2 instance using Git Bash from a Windows machine, install Apache, and serve a custom website.

## 🚀 What I Did
- Launched an EC2 instance using AWS Console
- Connected via Git Bash using `.pem` key
- Installed Apache2 on Ubuntu EC2
- Served a custom HTML page
- Documented steps, screenshots, and common errors

## 📂 Project Structure
See folders:
- `ec2-instance-setup/`: Bash scripts and EC2 launch notes
- `web-server-setup/`: Apache install script and sample website
- `screenshots/`: Proof of work with EC2 and webpage running
- `notes/`: Error logs and troubleshooting
